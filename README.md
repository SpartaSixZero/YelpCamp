# YelpCamp
YelpCamp is a Full Stack web application that I created for a hobby as a part of a Udemy course that I took, The Web Developer Bootcamp 2020. This project leverages the following technologies:

- MongoDB
- Mongoose
- Express
- EJS for templating
- JavaScript
- HTML/CSS
